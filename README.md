# Unity-3D-Geometry-Learning-App

An educational application developed using **Unity** that helps students learn **3D geometric shapes (solid geometry)** through **interactive Augmented Reality (AR)** experiences.

This application is designed to make learning geometry more engaging by visualizing 3D shapes in real space, displaying mathematical formulas, and providing quizzes to test user understanding.
